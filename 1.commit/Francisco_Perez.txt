PREGUNTAS TEORICAS:
¿Cómo se inicializa un repositorio local?
git init

¿Cómo hago para que un directorio deje de ser controlado por git?
rm -rf .git
Si agrego un archivo a un directorio que ya está siendo controlado por git, ¿está siendo controlado por git? No automáticamente. Para que el archivo sea controlado por git, debes agregarlo con el siguiente comando:
git add <nombre_del_archivo>

¿Qué comando se utiliza para agregar un archivo al repositorio local?
git add <nombre_del_archivo>
Para agregar todos los archivos modificados o nuevos:
git add .

¿Cómo determino que archivos fueron modificados?
git status

¿Qué comando se utiliza para hacer un commit?
git commit -m "Tu mensaje de commit"

En sus propias palabras, ¿qué es un commit? 
Un commit es un registro que guarda los cambios realizados en el repositorio local.
